About Push Notifications

Push notifications allow users to receive real-time updates even when the app is closed or in the background. VAPID keys are required to send push notifications securely.

How It Works

  • 1. Generate VAPID keys using the button above
  • 2. Users will automatically be prompted to allow notifications when they log in
  • 3. Notifications are only sent when the app is closed or in the background
  • 4. When the app is active, the existing polling system is used instead
